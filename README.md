## About

I'm George Nădejde, an aspiring cyber security professional currently pursuing an MSc in Cyber Security at Radboud University. I have a strong interest in how systems work, break, and can be secured — from low-level hardware to network protocols and web applications.

I specialize in:
- Penetration testing & vulnerability research
- Reverse engineering & digital forensics
- Secure software development

I actively sharpen my skills through platforms like TryHackMe and Hack The Box, and I'm open to opportunities in offensive or defensive security roles.

## Key Features

- Terminal-aesthetic dark UI with electric green and cyan accents
- Animated typewriter hero sequence
- Scroll-reveal animations and animated skill bars
- Floating particle canvas background
- Fully responsive — works on desktop and mobile
- Contact form powered by Formspree

---

## Technologies

**Frontend**
- HTML5, CSS3 (custom animations, responsive layout, CSS variables)
- Vanilla JavaScript
- Google Fonts — JetBrains Mono & Syne

**Cyber Security Tools Featured**
- Metasploit, Wireshark, BurpSuite, Nmap, Ghidra, Frida

**Deployment**
- GitHub Pages with automated CI/CD via GitHub Actions
